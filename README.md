# Line Tracking Robot With Arduino And Freertos
Implementation of a smart line-tracking robot with the following features:
- Distance detector
- Photoresistor
- Tilt sensor
- PID speed control

You can also read more in the [report file](/EMB-Final%20Project-Report.pdf).
